# Frontend - React Application

This is the frontend for the PTA & RTA Permit Management System built with React.

## Installation

```bash
cd frontend
npm install
```

## Environment Variables

Create a `.env` file in the frontend directory:

```
REACT_APP_API_URL=http://localhost:8000
```

## Running the Development Server

```bash
npm start
```

The application will open at `http://localhost:3000`

## Building for Production

```bash
npm build
```

## Project Structure

- `public/` - Static files
- `src/` - Source code
  - `components/` - Reusable React components
  - `pages/` - Page components
  - `services/` - API service and utilities
  - `styles/` - CSS stylesheets
  - `App.js` - Main App component
  - `index.js` - Entry point

## Features

- Dashboard with permit statistics
- Permit management (list, create, edit, cancel)
- Filter permits by status
- Form validation
- API integration with Django backend
- Responsive design

## Dependencies

- React 18
- React Router DOM
- Axios for HTTP requests

## Available Scripts

- `npm start` - Start development server
- `npm build` - Build for production
- `npm test` - Run tests
