## ‚ú® Key Concepts

### What's New
- **3 Models**: Feature, Role, UserRole
- **18 Endpoints**: User, Role, Feature management
- **4 Roles**: Admin, Operator, Supervisor, End User
- **13 Features**: Fine-grained permissions
- **2 Pages**: User & Role management

### How It Works
1. **User** has a **Role**
2. **Role** has **Features**
3. **Features** control what user can do
4. **Admin** manages users and roles

### Key Terms
- **Feature**: Individual permission
- **Role**: Group of features
- **UserRole**: Link between user and role
- **RBAC**: Role-Based Access Control
- **Permission**: Ability to do something
